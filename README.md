# η = 0.116: Spacetime-to-Time Conversion in Black Holes

## Discovery
Empirical constant η ≈ 0.116 discovered from EHT data analysis.

## Key Results
- M87*: Perfect match with EHT (42.0 μas predicted vs 42.0 μas observed)
- Sgr A*: Within error margins (53.8 μas vs 51.8 ± 2.3 μas)
- GR underestimates by ~5.8%

## Formula
Effective horizon radius: r_h = r_s × (1 + η/2)
Shadow diameter: θ = θ_GR × (1 + η/2)

## Predictions
- NGC 4258: 10.2 μas
- Centaurus A: 23.4 μas
- M31*: 56.8 μas

## Interpretation
η represents the fraction of spatial structure converted to temporal structure at event horizons (~11.6%).
